# LibreStore

O **LibreStore** é um cliente alternativo para o repositório **F-Droid**, desenvolvido com foco total em **acessibilidade** e simplicidade.

## 🚀 Principais Funcionalidades
- **Acessibilidade Nativa:** Integração total com Text-to-Speech (TTS).
- **Navegação por Categorias:** Catálogo organizado e traduzido.
- **Busca Inteligente:** Encontre apps rapidamente.
- **Gerenciador de Downloads:** Instalação direta de APKs.
